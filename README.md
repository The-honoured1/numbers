# numbers

**numbers** is a premium mobile mini-game collection inspired by the clean, minimal, and playful aesthetic of **NYT Games**. Designed for logic and math enthusiasts, it offers a variety of brain-teasing puzzles wrapped in a sophisticated, animated interface.

---

## 🎮 The Collection

1.  **Number Grid (Sudoku)**: Classic 9x9 logic puzzles with three difficulty levels (Easy, Medium, Hard).
2.  **2048**: A smooth, gesture-based 4x4 sliding tile game. Merge numbers to reach the legendary 2048 tile.
3.  **Math Puzzle**: A fast-paced, timed quiz challenge. Solve dynamic equations that scale in difficulty.
4.  **Sequence**: Reveal the hidden logic in Arithmetic, Geometric, Fibonacci, and power-based number sequences.
5.  **Countdown**: Use 6 random numbers and operators (+, -, ×, ÷) to reach a specific target within 60 seconds.
6.  **Math Cross**: A unique 2D crossword where row and column equations must be solved simultaneously.

---

## ✨ Features

- **NYT-Inspired Design**: Clean white backgrounds, vibrant accent colors, and sophisticated serif typography (Lora & Inter).
- **Premium Animations**: Staggered entrance motions, elastic "bounce" UI elements, and shimmering interaction cues.
- **Daily Persistence**: Offline streak tracking and high scores powered by `shared_preferences`.
- **Fully Offline**: Play anywhere, anytime without an internet connection.
- **Responsive Logic**: Puzzles are dynamically generated to ensure endless replayability.

---

## 🛠 Tech Stack

- **Framework**: [Flutter](https://flutter.dev/)
- **State & Logic**: Dart
- **Animations**: `flutter_animate`
- **Typography**: `google_fonts`
- **Storage**: `shared_preferences`

## 🚀 Getting Started

### Prerequisites

- Flutter SDK installed.
- An IDE (VS Code, Android Studio) or terminal access.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/The-honoured1/numbers.git
    cd numbers
    ```

2.  **Install dependencies**:
    ```bash
    flutter pub get
    ```

3.  **Run the app**:
    ```bash
    flutter run
    ```
